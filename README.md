============================================================

This project hosts Python implementations of the Info fuzzy network(IFN) algorithm.

## Dependencies ##

* numpy
* scipy
* scikit-learn

## How to use ##
Download, import and do as you would with any other scikit-learn method:
* fit(X, y)
